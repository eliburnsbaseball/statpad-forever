# StatPad

## Deploy to Vercel (Recommended - Free, ~3 minutes)

1. Download this folder as a zip and extract it
2. Go to [vercel.com](https://vercel.com) and sign up (free)
3. Click "Add New Project" → "Import Git Repository"
   - OR: drag the folder into Vercel's drop zone at vercel.com/new
4. Vercel auto-detects Vite — just click **Deploy**
5. Your game is live at `yourproject.vercel.app`

## Deploy to Netlify (Also Free)

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag the entire `statpad-deploy` folder onto the page
3. Wait ~30 seconds → live URL

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173
